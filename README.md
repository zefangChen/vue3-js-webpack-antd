# 项目说明文档

## node环境说明

> node.js版本  v14.16.0及以上
>
> npm 版本  6.14.11及以上

## 本地开发命令
### 第一步：依赖安装
```
npm install
```

### 第二步：项目启动
```
npm run serve
```

### 生产打包命令
```
npm run build
```
